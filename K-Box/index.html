K-Box | jQuery Plugin

Plugin Overview

K-Box is a simple modal box that makes easy popup boxes for your site. No IE fix yet, but coming soon, I'm just too lazy to make one for it. :p

Examples

Use an anchor tag to invoke the box, the reference must include #kb-init. Initialize K-Box

Plugin Specifications

The box is invoked when the anchor link is clicked, and the box overlays the entire page. It's not one of those annoying auto-popup boxes, though that will be added in a later version, and an option to make it that way or not.
To obtain, grab the code from Github or direct download.

How to Use It

All you need to do is include the javascript file, the images (for close button) and the stylesheet: 

<script type="text/javascript" src="./js/kbox.min.js"></script>
<link rel="stylesheet" type="text/css" href="./css/kbox.css">

Now, just make the anchor link and the box itself with a simple div element. Give the div the title attribute to give your box a title: 

<a href="#kb-init">invoke</a>
<div name="kb-modal" title="Modal Box Title">Content that goes within the modal box</div>

You do however need jQuery to use this, as it is a jQuery plugin.
